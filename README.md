# dual-mode-sensor
A smart ESP32 weather station that shows real-time temperature and humidity on both an OLED screen and a live web page. When it gets too hot (over 28°C), it lights an LED as a warning. I added a new featureto it: Press the button to toggle between sensor readings (temperature/humidity) and network info (IP address + uptime). The display switches instantly on press and returns to sensor mode when released.
